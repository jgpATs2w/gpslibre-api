# starter-api

## Installation

```
git clone https://github.com/jgpATs2w/starter-api/
mv starter-api your-project-name
cd your-project-name
rm -rf .git
```

## Design principles  
  - Standars: PSR, HTTP
  - TDD (integration & testing)
  - Documented  
  - [API standards](apistandards)

## Requirements
  - Linux server
  - Apache
  - MySQL 5.3
  - PHP 5.5
